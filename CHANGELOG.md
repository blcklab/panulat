# Changelog

## 0.1.0 - Initial Release

First public release of the Panulat starter project.

### Added

* Starter project for `composer create-project blcklab/panulat my-api`
* REST API application structure
* Health and readiness endpoints
* Example users CRUD routes
* JWT register/login example
* Protected `/v1/me` route example
* Validation examples
* Database migrations and seeders
* SQLite local setup
* Docker MySQL setup
* Request ID middleware
* JSON, form, and multipart request support
* Beginner documentation in `docs/getting-started.md`

### Included Packages

* `blcklab/panulat-core`
* `blcklab/panulat-jwt`
* `blcklab/panulat-cli`
